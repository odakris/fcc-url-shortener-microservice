URL Shortener Microservice

This is an API that shorten a given valid URL

URL Shortener

Get shortened url

Provide a valid URL such as 'https://freecodecamp.org/' and get the shortened URL :

{ "original_url": "https://freecodecamp.org/", "short_url": "peSqXI0pC" }

Get redirected to original url

It is also possible to be redirected to the original URL such as :

[your_url]/api/shorturl/<short_url>

with <short_url> as a valid shortened URL like 'peSqXI0pC' :

[your_url]/api/shorturl/peSqXI0pC
